# CapsLockにCtrlを割り当てる

[Ctrl2cap](https://technet.microsoft.com/en-us/sysinternals/bb897578.aspx)を使う  

ダウンロードしてexeを実行する  
管理者権限が必要なので、コマンドプロンプトを右クリック、管理者権限で実行する  

```
> ctrl2cap.exe /install
```
